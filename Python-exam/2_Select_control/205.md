# 205 キャラクターコード判断

## プログラム説明：
以下の条件を満たすプログラムを作成せよ。ユーザーが文字を入力し、大文字と小文字を含む英字（アルファベット＝alphabet）、数字（数字＝number）、またはその他の文字（記号＝symbol）が含まれているかどうかを判別できるようにします。 例：aは英字、9は数字、$はその他の文字です。


## 入出力：
入力説明：

一つの文字

表示説明：

判断は英字（大文字と小文字を含む）、数字、またはその他の文字です

## 入出力例
```
入力例1
P
出力例1
P is an alphabet.

入力例2
@
出力例2
@ is a symbol.

入力例3
7
出力例3
7 is a number.
```
