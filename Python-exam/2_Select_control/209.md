# 209 距離の判断

## プログラム説明：

以下の条件を満たすプログラムを作成せよ。
ユーザーに平面座標値xとyを入力させ、その座標が座標（5, 6）との距離が15以内にあるかどうかを判断する。15以内であれば【Inside】と表示し、そうでなければ【Outside】と表示すること。

補足：平面上の2つの座標距離の計算公式：
![](https://i.imgur.com/drYFUWZ.png)


## 入出力：
入力説明：

数値xとy

出力説明：

距離15以下であればInside、15を超えた場合Outside

## 入出力例
```
入力例1
7
20
出力例1
Inside

入力例2
30
35
出力例2
Outside
```
