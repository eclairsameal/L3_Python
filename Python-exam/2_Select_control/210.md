# 210 三角形判断

## プログラム説明：

以下の条件を満たすプログラムを作成せよ。ユーザーに３つの辺を入力させ、それらの辺で１つの三角形として成り立つかどうかを判断すること。もし成り立つ場合は三角形の周りの長さの値を、成り立たない場合は【Invalid】と表示すること。

補足：三角形の判断方法　→　３辺のうち、２辺の長さの合計が残りの１辺より長くならなければならない。


## 入出力：
入力説明：

自然数3つ

出力説明：

三角形として成り立てば周りの長さを、出なければInvalidを表示させる

## 入出力例
```
入力例1
5
6
13
出力例1
Invalid

入力例2
1
1
1
出力例2
3
```
