# 成績計算

## プログラム説明：

次の条件を満たすプログラムを作成せよ。
ユーザーに生徒3名それぞれ5つの成績を入力させ、生徒一人一人の成績の総和と平均を出力し表示すること。

ヒント：平均は小数点以下2桁まで表示させること。


## 入出力：
入力説明：

生徒3名分のそれぞれ5つの成績

出力説明:

規定フォーマットで生徒一人一人の成績の総和及び平均の出力表示
