# 610 平均温度

## プログラム説明：

以下の条件を満たすプログラムを作成せよ。

ユーザーに3日間の気温を4週間分入力させ、そのデータを元にこの4週間の平均気温、最高気温、そして最低気温を表示すること。

ヒント1：平均気温の表示時、小数点以下2桁までとする
ヒント2：最高気温及び最低気温の出力は次の例に従って出力する。例：「31」であれば「31」と、「31.1」であれば「31.1」と出力する

##  入出力：
入力説明:

1週間3日分の気温データ、これを4週間分

出力説明:

平均気温

最高気温

最低気温
