# 103 文字列を指定したフォーマットで表示

## プログラム説明：

次のプログラムを作成せよ。
4つの英単語を入力し、これらの単語に10スペースを用い、単語同士の間に1つスペースを置く。
列ごとに2つの単語を表示し、先に右揃えで表示した後に、左揃えで表示させること。
列の始めと終わりは、|（バーティカルバー）を用いて境界とすること。

## 入出力：
入力説明：4つの英単語

表示説明：指定したフォーマットで表示

## 入出力例
```
入力例
I
enjoy
learning
Python

表示例
|         I      enjoy|
|  learning     Python|
|I          enjoy     |
|learning   Python    |
```
