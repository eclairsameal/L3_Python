# 102 小数点入りの数字を指定フォーマットでの表示

## プログラム説明：

次のプログラムを作成せよ。
以下のプログラムを作成せよ。 ４つの1～４桁の小数を含む数字を入力し、これらの数字を７つのスペースを用い、数字と数字の間にスペースを1つ空けておくこと。 そして、列ごとに２つの数字を表示し、まず右揃えで４つの数字を表示させ、次に左揃えで４つの数字を表示させること。 列の始めと終わりは、|（バーティカルバー）を用いて列の境界とすること。

## 入出力：
入力説明:4つの小数を含む数字

表示説明:指定フォーマットでの表示

## 入出力例
```
入力例
23.12
395.3
100.4617
564.329

表示例
|  23.12  395.30|
| 100.46  564.33|
|23.12   395.30 |
|100.46  564.33 |
```
