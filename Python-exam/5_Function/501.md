# メッセージの表示

## プログラム説明：

次の条件を満たすプログラムを作成せよ。
関数「compute()」を作成し呼び出すようにする。関数の機能はユーザーに学部（Department）、学生番号（Student ID）、氏名（Name）を入力させ、これを出力し表示させること。

## 入出力：
入力説明:

3つの文字列

出力説明:

学部（Department）

学籍番号（Student ID）

氏名（Name）

## 入出力例

```
入力例
Information Management
123456789
Tina Chen
出力例
Department: Information Management
Student ID: 123456789
Name: Tina Chen
```
